Trying Numpy for real

Trying Numpy for real#

Numpy is just a Pythong library. So, we can clearly use it on any Jupyter Notebook. However, you can also try it directly on their website. They have some sort of sandbox that you can write code directly without any installation on your local machine. see Fig. 3

To try numpy first of all, you need to import the package first. This can be installed by the following command, if you use conda

conda install numpy

And then we need to import it first.

import numpy as np

We also need to import pyplot to demonstrate how to use numpy to plot some graphs.

import matplotlib.pyplot as plt

Referring to for the the (1) on the What is numpy, you asked. We modify a bit to make it more complicate to be:

(3)#\[x=\sin(\frac{2\pi}{2})\]

If we want to find the value of x, we can code as follow:

x = np.sin((2*np.pi)/2)
x

1.2246467991473532e-16

And for the (2) we can write code like this.

np.tan(x) == np.sin(x)/np.cos(x)

True

And following is how we can use numpy to plot the sine graph.

import matplotlib.pylab as plt
x = np.linspace(-np.pi, np.pi, 201)
plt.plot(x, np.sin(x))
plt.xlabel('Angle [rad]')
plt.ylabel('sin(x)')
plt.axis('tight')
plt.show()

So, it is no brainer that we can modify the above code to plot graph for cosine. We can plot cos graph by use the followed code:

x = np.linspace(-np.pi, np.pi, 201)
plt.plot(x, np.cos(x))
plt.xlabel('Angle [rad]')
plt.ylabel('cos(x)')
plt.axis('tight')
plt.show()

Let me give you another example, on (4)

(4)#\[\arccos x=\cos^{-1} x\]

This is inverse cosine(arccos) graph.

x = np.linspace(-1, 1, num=100)
plt.plot(x, np.arccos(x))
plt.axis('tight')
plt.show()

We can prove (4) by following code.

if np.arccos(np.cos(np.pi)) == np.pi :
    print("It's true")
else:
    print("It's not true")

It's true

By the way if you are not sure what is sine, cosine, tangent. It is one of the mathematic thing called “Trigonometry”. Khan Acadely has a great explanation about it.

x = np.linspace(0,np.pi*2,100)
plt.plot(np.cos(x), np.sin(x), linewidth=1)
