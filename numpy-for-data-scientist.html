Recap#

Just in case that you are curious what is mathematic formula for Fig. 4 of the former chapter.

(5)#\[\sin^2\theta + \cos^2\theta = 1\]

(5) is called Pythagorean formula for sines and cosines. This is probrably the most important identity according to this website from Clark Univeristy. But if you are interested on formula that has important name in it. Those followings are called “Ptolemy’s identities”.

More formula for Trigonometry

Please visit Dave’s site. There is a compilation there!

(6)#\[\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta\]
(7)#\[\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta\]
(8)#\[\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta\]
(9)#\[\cos(\alpha - \beta) = \cos\alpha\cos\beta + \cos\alpha\sin\beta\]

Numpy with Pandas#

No no.. This is not about Fig. 5 but aboout a library pandas that uses by data scientist to deal with huge amount of data.

Pandas introduce the concept of dataframe and operation on it to facilitate the analysis of big data.

df = pd.DataFrame({"A":[12, 4, 5, 44, 1],
                   "B":[5, 2, 54, 3, 2], 
                   "C":[20, 16, 7, 3, 8],
                   "D":[14, 3, 17, 2, 6]})

From the code above, the data frame is created with 4 columns: A, B, C and D. as shown in Fig. 6.

We can calculate mean of each column with code which return result as Fig. 7

df.mean(axis = 0)

Learn more about Pandas

Code and pictures explaining here is actually from geeksforgeeks.org. Please visit them to learn more!

You can see that it’s quite convinient to find the means or multiple columns on just one command. This is why Pandas is the essential tools for data scientists.
